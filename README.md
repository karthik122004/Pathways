# pathways

pathways is an educational iOS app that helps students understand how instructions move through a single‑cycle MIPS processor datapath. It is being developed as a course project at Texas State University.

## Project Overview

The goal of this project is to provide an interactive learning aid for computer organization students. The app will visualize the datapath for a small subset of MIPS instructions and include a quiz component to support self‑assessment.

## Tech Stack

- Language: Swift  
- Framework: SwiftUI  
- Platform: iOS  
- IDE: Xcode

## Getting Started

1. Clone the repository.
2. Open the Xcode project file in the `pathways` directory.
3. Select an iOS simulator or device and build/run from Xcode.

More detailed setup and usage instructions will be added as the project progresses.
